# todo-cli-app
A simple command-line to-do list app built with Python.
## Description

This is a simple command-line To-Do List application built with Python. It allows users to add, view, and delete tasks directly from the terminal. The program stores tasks in a file, so they are saved even after the application is closed.

This project is designed for beginners to practice core programming concepts such as working with lists, handling file input/output, using loops for menus, and organizing code with functions.
